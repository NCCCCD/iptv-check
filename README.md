# IPTV播放列表检测

## 项目简介
1. 项目名称：IPTV播放列表检测
2. 项目代号：iptv-check
3. 一句话定义：检测 IPTV M3U 播放列表中每条流的可达性和播放效果

## 当前文件
1. `AGENTS.md`：项目级规则
2. `README.md`：项目入口说明
3. `docs/PROJECT.md`：项目唯一主文档
4. `修改意见.md`：当前仍有效的修改意见整合稿
5. `iptv-check.py`：初始脚本（第一版草稿）

## 下一步
1. 在该项目目录开执行线程
2. 在项目对话里发送 `1`，让系统先读取 `docs/PROJECT.md`
3. 提修改意见时，系统默认更新项目根目录下的 `修改意见.md`
4. 将 `docs/PROJECT.md` 作为唯一主文档继续推进
